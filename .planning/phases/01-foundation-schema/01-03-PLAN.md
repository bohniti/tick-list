    **UIAA to French conversion (per D-01, D-02):**
    - normalize_to_french("5+", "uiaa") == "6a+"
    - normalize_to_french("6-", "uiaa") == "6b"
    - normalize_to_french("7", "uiaa") == "7a"
    - normalize_to_french("8-", "uiaa") == "7b"
    - normalize_to_french("8+", "uiaa") == "7c"
    - normalize_to_french("9-", "uiaa") == "7c+"

    **UIAA slash grades (per D-04):**
    - normalize_to_french("7+/8-", "uiaa") == "7a+"
    - normalize_to_french("8+/9-", "uiaa") == "7c+"
    - normalize_to_french("6/6+", "uiaa") == "6b+"

    **YDS to French conversion:**
    - normalize_to_french("5.10a", "yds") == "6b"
    - normalize_to_french("5.11a", "yds") == "7a"
    - normalize_to_french("5.12a", "yds") == "7c"
    - normalize_to_french("5.9", "yds") == "6a+"

    **Auto-detection:**
    - normalize_to_french("5.10a") detects YDS automatically (starts with "5.")
    - normalize_to_french("7+") detects UIAA automatically
    - normalize_to_french("6b+") returns "6b+" unchanged (already French)

    **Numeric scoring (per D-01, monotonically increasing):**
    - grade_to_numeric("5a") < grade_to_numeric("5b") < grade_to_numeric("6a")
    - grade_to_numeric("6a") == 4.0 (per blog's gradeMap)
    - grade_to_numeric("7a") == 5.5
    - grade_to_numeric("8a") == 7.0
    - All consecutive French grades have strictly increasing numeric values

    **French slash grade scoring:**
    - grade_to_numeric("6a/b") == midpoint of grade_to_numeric("6a") and grade_to_numeric("6b")

    **Bouldering grades (per D-03 — separate axis):**
    - boulder_grade_to_numeric("6A") returns a score on the boulder axis
    - Route and boulder numeric scores are NOT comparable across axes
    - Font grades: 4, 5, 5+, 6A, 6A+, 6B, 6B+, 6C, 6C+, 7A, 7A+, 7B, 7B+, 7C, 7C+, 8A, 8A+
    - V-scale: V0-V16 mapped to Font equivalents

    **Edge cases:**
    - Unknown grades return 0.0 for numeric, raw string for normalize
    - Whitespace is stripped: normalize_to_french("  7+  ") == "6c+"
    - The "7/A" aid notation (Pitfall 5): returns the free climbing grade "7a" for the "7" component

    **Round-trip guarantee:**
    - For every grade in UIAA_TO_FRENCH: normalize then score produces a nonzero numeric value
    - For every grade in YDS_TO_FRENCH: normalize then score produces a nonzero numeric value
